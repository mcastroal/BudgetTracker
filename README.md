# BudgetTracker
A simple budget tracker application that can help you manage your income and expenses.

## How to Use
1. View Your Summary:
- The top table displays your total income, total expenses, and net income.

2. Add Income:
- Scroll to find the Income Table
- Enter a description (e.g. "Pension")
- Enter the amount
- Click "Add Income"
- The new incomes will appear in the table and your totals will update automatically

3. Add Expenses:
- Scroll to find the Expenses Table
- Enter a description (e.g. "Phone Bill") 
- Enter the amount
- Click "Add Expense"
- The new expenses will appear in the table and your totals will update automatically

4. Edit Existing Items:
- You can edit any description or amount directly in the table
- The totals update automatically as you type

## Features
- Real-time calculation of totals
- Input validation (prevents empty descriptions and invalid amounts)
- Mobile-responsive Design

## Technologies Used
- HTML5
- SASS/CSS3
- JavaScript (ES6 Classes)
